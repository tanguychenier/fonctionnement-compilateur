# Security Policy

## Reporting a Vulnerability

If you think you have found a security issue in this project, please open a private security advisory:

https://github.com/Tan-Software/fonctionnement-compilateur/security/advisories/new

Please do not open a public issue for vulnerabilities. We will acknowledge your report as soon as possible and coordinate a fix and disclosure timeline with you.

## Supported Versions

Only the latest version published on the `main` branch is actively supported.
